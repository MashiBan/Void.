Welcome to Void, a website dedicated to providing support and solace to individuals struggling with depression and emotional distress. Amidst life's storms, find a gentle refuge here where you can breathe freely and know that you're not alone.

## About

Void is a community-driven website designed to offer support and resources for individuals experiencing depression. Our platform provides a safe and supportive space where users can connect with others, share their experiences, and find comfort in knowing that there are people who understand.

## Features

- **Support Groups**: Join support groups tailored to different aspects of depression, such as coping strategies, self-care, and managing relationships.
  
- **Anonymous Discussions**: Engage in discussions anonymously to express your thoughts and feelings without fear of judgment.
  
- **Resource Library**: Access a wealth of resources including articles, videos, and other helpful materials to support your mental well-being.

- **Powered by Three.js and WebGL**: Void is built using Three.js and WebGL technologies, providing an immersive and interactive experience for users.

## Tech Stack Used

- **Frontend**: React.js, Three.js, WebGL
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Other Services**: Built with the MERN (MongoDB, Express.js, React.js, Node.js) stack

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
